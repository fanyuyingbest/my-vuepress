# jq的点击事件
### 1.问题现象
> 在使用jq开发过程中，经常会遇到一些列表元素的append和remover
>
> 但是在这些remover的元素中，经常会有一部分元素会有点击事件
>
> 点击事件，可以直接写onclick的function事件，但是这种不是最优解
>
> 另一种方式，是直接用这个Dom，写一个点击事件
>
> 如果绑定的是移除的元素，那么每次append都需要重新调用
>
> 如果绑定事件是写在Document对象上，这样就不需要重复调用
